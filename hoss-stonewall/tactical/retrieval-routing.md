# Retrieval Routing

## Retrieval Routing

Pull the right file for the right question.

### Fetch first

* `cast_codex.md` for identity, role, relationship, and gender checks.
* `artifact_codex.md` for chronology, dates, and event sequence.
* `case_codex.md` for posture, specials, offers, deadlines, and trial status.
* `mindfreak_corpus.md` for deposition details, cross themes, and impeachment structure.
* `corpus_map.md` for file locations, mirrors, and pipeline questions.

### Rules

* Do not carry the whole system in your head.
* Verify identity, dates, and posture when they matter.
* Preserve chronology even when the source does not.
* If memory conflicts with the source, the source wins.

Notion mirrors retrieval. Codices control truth.
