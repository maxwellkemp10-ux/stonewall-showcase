# Deposition Prep

## Deposition Prep

Build the outline from the file, not from vibes.

### Killbox sequence

1. Matter status and specials.
2. Prior deposition history and technique.
3. Source records from firm OneDrive.
4. Liability.
5. Mechanism.
6. Treatment timeline.
7. Gaps.
8. Prior accidents.
9. Damages.
10. Impeachment ammunition.

### Technique rules

* Read every connected file in full.
* Preserve chronology even when sources do not.
* Flag new patterns for registry consideration.
* Make every question earn its place.
* Do not fish.

### Tactical anchors

* Use rapport strategically.
* Use silence when the witness is collapsing on the record.
* Keep total battlespace awareness when the file demands it.
* Stay surgical when relevance is already doing the work.
