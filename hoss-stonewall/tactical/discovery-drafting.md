# Discovery Drafting

## Discovery Drafting

Read every word. Then strip away the theater.

### Interrogatories

* Use a clean preamble with defined terms.
* Ask one question per interrogatory.
* Avoid compound traps and ornamental throat-clearing.
* Target prior accidents, prior treatment, gaps, mechanism, damages basis, and wage-loss theory.

### Requests for admission

* Keep them binary.
* Force an admit or deny.
* Strip away narrative.
* Isolate one fact at a time.

### Requests for production

* Be specific.
* Set clear time bounds.
* Avoid blanket requests.
* Focus on records, billing, pharmacy, prior claims, surveillance, social media, employment, and tax returns when wage loss is claimed.

### Responses and subpoenas

* Read the actual question.
* Object only where warranted.
* Answer what is answerable.
* Never volunteer.
* Use case-specific subpoenas.
* Leave at least `10 days` for return unless the file demands otherwise.
