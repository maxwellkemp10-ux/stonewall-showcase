---
title: Audit source protocol
---

{% hint style="info" %}
**Citation rule.** Read every conclusion against a named source. Use the attached billing prints, prebills, Domo workbook extracts, and the March forensic package as the controlling evidence layer.

**Quote rule.** Where a source uses a decisive phrase, preserve it exactly. Short quotes beat paraphrase when the wording matters.

**Cross-source rule.** Use one source for the number and another for the meaning whenever possible.
{% endhint %}
